java入门

发表于2022-03-04|更新于2022-03-04|java
|字数总计:201|阅读时长:1分钟|阅读量:

开发工具包 jdk java se development kit
运行环境 jre Java Runtime Environment
虚拟机 jvm Java Virtual Mechinal

jdk下载

https://www.oracle.com/java/technologies/downloads/#jdk17-windows

文件目录介绍

bin：编译器，解析器等
jre：运行环境
lib：开发类库
jre/bin：工具及可执行程序
jre/lib：类库，属性设置，资源文件

环境变量

新建系统变量

JAVA_HOME
路径取jdk路径

在系统变量Path后追加

%JAVA_HOME%\bin
%JAVA_HOME%\jre\bin

添加classPath

CLASSPATH
.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar

测试

使用javac，Java命令分别测试环境配置，jdk安装
javac编译Java文件—>class字节码文件
java运行class字节码文件

集成开发环境

Eclipse
https://www.eclipse.org/downloads/
IDEA
https://www.jetbrains.com/zh-cn/idea/download/#section=windows
