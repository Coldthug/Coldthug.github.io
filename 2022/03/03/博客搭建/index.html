hexo命令

新建文件夹blog

博客初始化

本地启动博客服务

GitHub新建仓库

登录Github，new repository
repository name 为 昵称名.github.io
例如：我的呢称名为 Coldthug 对应为 coldthug.github.io
切记：一定是昵称名，切勿随意修改

安装git部署插件

配置_config.yml

_config.yml 为重要配置文件，切勿随意修改
详细请参考官网 https://hexo.io/zh-cn/docs/configuration

打开配置文件，在尾部修改

#branch（github默认为main）

博客部署

使用git部署，部署前配置git的用户名和邮箱地址
两条命令分开运行

使用 hexo d 命令部署
弹出登陆界面，输入用户名及密码
成功后在GitHub会有你的博客文件

测试

浏览器输入 昵称名.github.io
查看博客是否正常

主题更换

主题下载

https://hexo.io/themes/

主题配置

_config.yml中找到theme
修改为下载的主题文件夹名称
重新加载博客

博客美化

根据下载的主题对应的github及说明文档对界面代码进行修改添加信息
